# 我的 Obsidian Vault

歡迎來到你的個人知識庫！

## 📚 結構說明

- **notes/** - 主要筆記目錄
- **assets/** - 圖片、PDF、附件
- **templates/** - 筆記樣板
- **.obsidian/** - Obsidian 自動生成的設定（不要手動修改）

## 🚀 快速開始

### 建立新筆記
使用以下語法連結到其他筆記：
- `[[筆記名稱]]` - WikiLinks
- `[[筆記#標題]]` - 連結到特定標題
- `[[筆記|顯示文字]]` - 自訂連結文字

### 標籤
使用 `#標籤` 來分類筆記，例如：
- `#工作` `#學習` `#專案`
- `#重要` `#待辦` `#參考`

### 呼叫區塊
> [!INFO] 提示
> 這是資訊區塊

> [!NOTE] 筆記
> 這是筆記區塊

> [!IMPORTANT] 重要
> 這是重要區塊
