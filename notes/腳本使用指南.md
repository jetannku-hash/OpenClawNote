## 🔧 腳本使用方法

### 筆記上傳到 OpenClawNote

#### 方法 1：使用 upload_notes_fixed.sh（推薦）

**修正的腳本**：使用正確的 `github.com` 端點

```bash
# 執行腳本
bash /home/terryku/.openclaw/workspace/scripts/upload_notes_fixed.sh
```

**特點：**
- ✅ 端點正確（`github.com` 不是 `api.github.com`）
- ✅ 避免網絡解析問題
- ✅ 完全自動化上傳

#### 方法 2：手動上傳到 GitHub（最可靠）

**如果腳本失敗：**

1. **訪問 GitHub 頁面**
   - URL: https://github.com/jetannku-hash/OpenClawNote

2. **點擊「Upload files」**
   - 位於頁面右上角

3. **拖放上傳**
   - 拖放 `/home/terryku/.openclaw/workspace/my_obsidian_vault/` 整個目錄
   - 包含所有筆記、樣板、指南

4. **填寫提交訊息**
   - Commit message: `初始化筆記資料庫`
   - 或直接使用預設訊息

5. **完成上傳**
   - 點擊「Commit changes」
   - 等待上傳完成

## 💡 使用建議

### 短期：手動上傳（推薦）

**優點：**
- ✅ 不依賴任何腳本
- ✅ 最簡單、最直觀
- ✅ 不會遇到 API 問題
- ✅ 可以看到上傳進度

### 長期：使用 Git + SSH（最可靠）

**優點：**
- ✅ 最可靠、最快速
- ✅ 支援增量推送
- ✅ 可以推送大量檔案
- ✅ 之後只需 `git add` 和 `git push`

**設定步驟：**
1. 生成 SSH 金鑰
   ```bash
   ssh-keygen -t ed25519 -C "terryku@github.com"
   ```

2. 添加公鑰到 GitHub
   - 訪問：https://github.com/settings/keys
   - 點擊「New SSH key」
   - 貼上 `~/.ssh/id_ed25519.pub` 的內容

3. 使用 Git 推送
   ```bash
   cd /home/terryku/.openclaw/workspace/my_obsidian_vault
   git init
   git add .
   git commit -m "初始化筆記倉庫"
   git remote add origin git@github.com:jetannku-hash/OpenClawNote.git
   git push -u origin main
   ```
