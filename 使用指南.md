# Obsidian Vault 使用指南

## 🚀 快速開始

### 在 Obsidian 中打開
1. 打開 Obsidian 應用程式
2. 點擊「開開資料夾作為 Vault」
3. 選擇：`/home/terryku/.openclaw/workspace/my_obsidian_vault`

### 在命令列中操作
```bash
# 設定預設 vault（只需做一次）
cd /home/terryku/.openclaw/workspace/my_obsidian_vault
obsidian-cli set-default "my_obsidian_vault"

# 確認設定
obsidian-cli print-default
```

## 📚 筆記樣板

本 Vault 包含以下樣板：

### 1. 每日筆記
- **路徑：** `templates/每日筆記.md`
- **用途：** 記錄每日待辦、想法、學習筆記
- **標籤：** #每日 #筆記

### 2. 專案筆記
- **路徑：** `templates/專案筆記.md`
- **用途：** 規劃專案、追蹤任務、記錄進度
- **標籤：** #專案
- **狀態：** 🟢 規劃中 / 🟡 進行中 / 🔴 阻塞 / ✅ 已完成

### 3. 學習筆記
- **路徑：** `templates/學習筆記.md`
- **用途：** 記錄學習內容、核心概念、重點筆記
- **標籤：** #學習 #筆記

### 4. 會議紀錄
- **路徑：** `templates/會議紀錄.md`
- **用途：** 記錄會議內容、行動項目
- **標籤：** #會議 #工作

### 5. 空白筆記
- **路徑：** `templates/空白筆記.md`
- **用途：** 自由筆記的基礎樣板
- **標籤：** #筆記

## 📝 基本語法

### WikiLinks（連結到其他筆記）
```markdown
[[筆記名稱]]           # 連結到筆記
[[筆記#標題]]         # 連結到特定標題
[[筆記|顯示文字]]     # 自訂連結文字
```

### 呼叫區塊
```markdown
> [!INFO] 提示
> [!NOTE] 筆記
> [!TIP] 小貼士
> [!IMPORTANT] 重要
> [!WARNING] 警告
```

### 文字格式
```markdown
**粗體** 或 __粗體__
*斜體* 或 _斜體_
~~刪除線~~
==高亮==
`行內程式碼`
```

### 標籤
```markdown
#工作 #學習 #專案
#重要 #待辦 #參考
```

## 🛠️ 使用 obsidian-cli 指令

```bash
# 搜尋筆記
obsidian-cli search "筆記"

# 搜尋內容
obsidian-cli search-content "重要事項"

# 建立筆記
obsidian-cli create "notes/新筆記" --content "內容"

# 移動筆記（自動更新連結）
obsidian-cli move "舊路徑.md" "新路徑.md"

# 刪除筆記
obsidian-cli delete "筆記.md"
```

## 💡 使用建議

1. **使用樣板建立筆記**
   - 複製 `templates/` 下的樣板
   - 重新命名為你的筆記

2. **使用雙向連結**
   - `[[筆記名稱]]` 會自動建立連結
   - 點擊可跳轉，雙擊可預覽

3. **定期整理**
   - 使用標籤分類
   - 建立索引筆記

4. **備份 Vault**
   - 可以使用 git
   - 或定期複製整個資料夾
